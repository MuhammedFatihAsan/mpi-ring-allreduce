# Ring Allreduce

This is a small C and MPI project.

The goal is to test data sync between MPI ranks. The project uses dummy data.

The project has two main parts:

- Sequential baseline
- Ring Allreduce

The project uses OpenMPI, MPI_Wtime, and gprof.

## Folder Plan

- `include`: header files
- `src`: C source files
- `scripts`: test and result scripts
- `results`: raw data and charts
- `report`: project reports

Build and run commands will be added with the project code.
